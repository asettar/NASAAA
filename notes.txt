html contains a lot of tags ex: <p> </p>, <body> </body> ..
    -each tagname can have multiple attributes
<p> </p> tag -> ignores space and newline(except if you use <br>)
<pre> </pre> -> like <p> tag bur it doenst ignore spaces and line breaks, and also it has fixed width font size
        it stand for preformatted text.

style attribute :
    syntax :  <tagname style:"property:value;">
        properties example:
        <h1 style="background-color:powderblue;">This is a heading</h1>
        <p style="background-color:tomato;">This is a paragraph.</p>
        <h1 style="color:blue;">This is a heading</h1>
        <h1 style="font-family:verdana;">This is a heading</h1>     // define the font of text 
        <p  style="font-size:160%;">This is a paragraph.</p> 
        <h1 style="text-align:center;">Centered Heading</h1>        // to center a paragraph 
    
to add CSS to html pages: 3 ways.
    most common to make it in separate .css file and then link in head section  uusing link tag ex: 
        <!DOCTYPE html>
            <html>
            <head>
            <link rel="stylesheet" href="styles.css">
            </head>
            <body>
    inline css : definle style attribute to a an html element ex:  
        <h1 style="color:blue;">A Blue Heading</h1>
    external Css : is used to define CSS of all the page, it is defined in the head section of html file. ex: 
        <!DOCTYPE html>
        <html>
        <head>
        <style>
        body {background-color: powderblue;}
        h1   {color: blue;}
        p    {color: red;}
        </style>
        </head> 

-Html links : it is used to jump from one document(page) to another. it can be a text or img or any html element .
    syntax : <a href="url">link text</a>
        -> linke text is the part that will be shown to the user,  url is the url of the new page that you want to go to 
        -> by default the linked page will be displayed in the current browser window, to change you must specify a target to it
        ex:  <a href="https://www.w3schools.com/" target="_blank">Visit W3Schools!</a> 
            taget blank means open it in a new window or tab.
        -> there is 2 types of urls: Absolute urls (a fulle new web address) in the href attribute.
            and relative links(links to a page withn the same website) without mentinonng https://www part 
        ->button is also  can be considered as a link
            syntax :  <button onclick="document.location='pagepath'">HTML Tutorial</button> 
        -> also create link as a bookmark:
            we define bookmark like this :  
                <h2 id="C4">Chapter 4</h2> //it will create a bookmark having id=C4, then you add a linkg 
                    when clique on it will move you to the head having name = chapter4 

                ex: <a href="#C4"> Jump to Chapter 4</a> 
        -> img as a link : ex  
            <a href="default.asp">

-Class attribute often used to point to a class name in a style sheet. for ex if there is 3 <div>
    -having class attricute class="city", all the 3 divs elements will be styled equally according
    to the .city style definition in css 
    - to create a classe -> .classname { }

-id attribute specify a unique id for an html element. it is used to point to a specific style 
declaration in a style sheet. in <style> id is defined with #id {...}
    ex : <h1 id="myHeader">My Header</h1> -> will have my header style
- the diff between id and class -> class can be used by multiple html element while id only one 
    html element within the page. 

-html bookmarks with id and links: bookmarks allow users to jump to a specific part of the webpage
    to use it you must first create it, then add a link to it when it is clicked it will jump to it 
    ex: 
         <a href="#C4">Jump to Chapter 4</a> 
        <h2 id="C4">Chapter 4</h2>
        or jump to id=C4 of another page 
            ex: <a href="html_demo.html#C4">Jump to Chapter 4</a>
    
-<meta> element is defined int the head of html, contains metadatam ex of meta attributes :
    <meta name="keywords" content="HTML, CSS, JavaScript">  : keywords for search engine 
    <meta name="description" content="Free Web tutorials">  : decsription of you web site 
    <meta name="author" content="John Doe">

        good on all devices.
